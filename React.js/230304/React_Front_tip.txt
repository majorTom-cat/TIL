외부 스타일 방식을 기본으로 생각하고있었지만 왜인지 모르게 적용이 안되어
부득이하게 내부 스타일 방식을 채택했다.
이후 캐러셀 관련 컴포넌트를 적용할때 외부 스타일이 잘 동작했다. 왜일까
오타였을까..?

html css로 요소들을 배치하는데에 수많은 스트레스와 시간이 나의 삶을 통과한다.
웹페이지 깎는 노인이 될 지경이다. 곧 흰머리가 자라나도 이상하지 않을것이다.

외부 스타일을 적용할땐
import "./styles/blablaStyle.css";

한 컴포넌트에만 적용하고 싶을땐 적용하고자 하는 컴포넌트에
import styles from "./style/DetailPicture.module.css";
한 뒤 className={styles.slide} 이런식으로 {} 안에 styles. 를 붙여 클래스 이름을 써준다.
비교 연산자 그리고 2개 이상의 클래스 적용에는
className={
              i === current
                ? [styles.dot, styles.current].join(" ")
                : [styles.dot].join(" ")
            }
처럼 join() 함수를 써준다.

클라이언트의 드래그 관련 css 속성은
user-select: none; -> 드래그 안되게함

구글링으로 캐러셀을 찾아 활용했다.

수많은 시행착오들을 글로 적고싶었으나 시간이 흘렀고 썻다 지웠다 한 여러 방법과 코드들을 적기엔 여백이 부족하다.
